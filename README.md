Conky configuration file
========================

This is my Conky configuration files. 


Installation
------------

Fetch the configuration files from GitHub repository:

``
$ git clone git://github.com/jyjeanne/conky_laptop.git ~/.conky
``

Create link:

``
ln -s ~/.conky/conkyrc ~/.conkyrc
``

Enjoy!

Features
--------
* OS version 
* kernel version
* CPU graph
* memory graph
* network graph
* processes
